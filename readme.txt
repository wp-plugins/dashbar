Enhance Wordpress Admin bar to add features present in previous versions of DashBar: mainly edit posts displayed on current page

== Description ==

This plugin tries to enable previous feature present in DashBar in the new Wordpress Admin Bar.

No more options, only contextual links missing in the admin bar:

* Choose from the list of posts displayed to edit one

Network admin is also accessible from the first menu

No more localization required as this plugin uses core WordPress text.

WARNING: This plugin requires PHP 5.


== Installation ==

Install from the repository or:

1. Upload the content of the directory DashBar to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
